What is The Accurate Hierarchy?
[
]
The Accurate Hierarchy
(or The Omni Hierarchy) is the expansion of
The Official Hierarchy
, instead of listing the most important and official objects, it tries list of every object of each class, this hierarchies will have only one class, and each class will be divided in levels. Because one accurate class will contain lots of objects. As well as every hierarchy, each class will have class terminator and chains.
For non-hierarchial list, See
here
