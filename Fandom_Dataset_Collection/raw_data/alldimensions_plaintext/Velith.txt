Definition
[
]
Velith (or ᖻ) is a number used to describe the limit of a verse’s capabilities in one aspect, depending on what page/verse this number is used in
“every verse has it’s own velith”
if you want to customize velith a bit, then there are three more variants
Variants
[
]
ᖻ
<
: The previous verse’s (the verse being contained by the current one) velith
ᖻ
>
: The next verse’s (the verse containing the current one) velith
ᖻ
(name of verse (NOV), shortened or not)
: (NOV)’s velith
