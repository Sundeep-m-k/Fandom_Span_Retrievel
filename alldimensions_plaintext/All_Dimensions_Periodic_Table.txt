Layers
[
]
Each layer is like its own periodic table. The layers stack on top of each other, creating a tower of layers. Here is a better description: There are flat plates that stack on top of each other to form a tower. I hope that was good enough.
ADPT Number Scale
[
]
The ADPT Number Scale Gives elements 2 numbers. Here is what you would write: the elemental number on the layer,the number of the layer. Example: if an element was number 6,069 on layer 6,657,808,898, it would be 6069,657808898 on the ADPT Number Scale.
Rows
[
]
Rows are rows of elements in layers that read left to right. They do not affect the ADPT Number Scale.
